
# Gym ERP Frontend

헬스장 직원·회원·PT 일정을 관리하는 Gym ERP 시스템의 **React 프론트엔드**입니다.  
트레이너별 PT 일정, 휴가, 기타 일정과 회원 정보를 한 화면에서 확인하고 관리할 수 있습니다.


---

## 사용 기술

- React (Vite)
- React-Big-Calendar
- Axios
- React-Bootstrap
- CSS (Custom, Bootstrap)

---

## 🧩 주요 화면·기능

### 1. 트레이너 일정 캘린더
- `react-big-calendar`를 활용한 주/월 단위 스케줄 조회
- 일정 유형별 색상 구분 (PT / 휴가 / 기타)
- 오늘 날짜 강조, 커스텀 툴바

### 2. 일정 등록 모달
- PT / 휴가 / 기타 일정 탭으로 구분된 모달 UI
- 회원 검색 모달을 통해 PT 대상 회원 선택

---

##  개인 기여 (윤재원)

- `ScheduleCalendar.jsx`, `ScheduleModal.jsx`,설계·구현
- 일정 등록/수정/삭제 API 연동 로직 작성
- 캘린더 UI 커스터마이징 (툴바, 오늘 강조, 시간대 설정 등)
